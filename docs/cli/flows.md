# `joopo tasks flow`

Flow commands are subcommands of `joopo tasks`, not a standalone `flows` command.

```bash
joopo tasks flow list [--json]
joopo tasks flow show <lookup>
joopo tasks flow cancel <lookup>
```

For full documentation see [Task Flow](/automation/taskflow) and the [tasks CLI reference](/cli/tasks).
